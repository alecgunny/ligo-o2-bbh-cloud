You can find a Google Drive folder containing some zipped initial data for the plots here. To recreate analysis.ipynb, which contains both the plots and some discussion on them, first built the conda environment described in environment.yaml in the root directory

conda env create -f environment.yaml

Then download data.zip from the linked drive and unzip to analysis/data. Then activate the conda environment and run the notebook from the analysis directory.

conda activate o2
cd analysis
jupyter notebook analysis.ipynb
